# Maker

MAKER ist keine Lern-App.
MAKER ist ein Instrument.

Ziel:
Menschen handlungsfähig machen.
Durch Stimme. Durch Struktur. Durch Wiederholung.

## Kernprinzip

- Kein Button
- Kein Scrollen
- Kein Konsum
- Handlung ist Pflicht

Start erfolgt über Stimme: "Start"

## Architektur

- Smart Linear Path
- 3 Kompetenz-Dimensionen:
  - rec (Rezeptiv)
  - prod (Produktiv)
  - ctx (Kontextuell)

Dieses Repository enthält die PWA-Version von MAKER.
